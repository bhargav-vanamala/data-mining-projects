# Logistic Regression Classification

## Objective
Build and evaluate a classification model using logistic regression.

## What This Project Covers
- Data preprocessing and encoding
- Train-test split
- Logistic Regression model training
- Confusion Matrix and Classification Report
- ROC Curve and AUC analysis

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Outcome
Developed a baseline classification model and evaluated performance using standard metrics.
